## Install Node.js

### Pre-check

1. Open `CMD` prompt and check if node is installed already
    - `node -v`
2. Ignore if you have relatively `latest` version (anything above `v20.x.x`)

### Steps

1. Download Node.js
    - [Download Node.js](https://nodejs.org/en/download/prebuilt-installer)
2. Click on `Download Node.js`
2. Run the installer

### Post-check

1. Check if node is sucessfully installed
    - `node -v` -> `vxx.xx.x`
    - `npm -v` -> `xx.x.x`
2. Check the install path
    - `which node`
3. General help option
    - `node --help`

To generate and view the Allure report, run your tests and then use the Allure command-line tool:

1.

Run your tests: npm run wdio

2.

Generate the report: allure generate allure-results --clean -o allure-report

3.

Open the report: allure open allure-report

How to Run Tests with Different Browsers:
You can now easily switch browsers using the command line:
•
Chrome (Default): npm run wdio
•
Firefox: BROWSER=firefox npm run wdio (On Windows: set BROWSER=firefox && npm run wdio)
•
Edge: BROWSER=edge npm run wdio (On Windows: set BROWSER=edge && npm run wdio)
